- HTML5, Tailwind CSS
- Responsive design

## 📁 Deployment Instructions
1. Clone or fork this repo
2. Push to a GitHub repo named `kaylemccarthy`
3. Enable GitHub Pages in repo settings → Pages → Source: `main` → `/root`
4. Visit your live site!
